# Тестовое задание 

### Задание

Сайт: wildberries.ru

Нужен каталог товаров по поисковому запросу "пальто из натуральной шерсти" в XLSX формате 

Список нужных данных:

- Ссылка на товар
- Артикул
- Название
- Цена
- Описание
- Ссылки на изображения через запятую
- Описание
- Все характеристики с сохранением их структуры
- Название селлера
- Ссылка на селлера
- Размеры товара через запятую
- Остатки по товару (число)
- Рейтинг
- Количество отзывов

Отдельным XLSX-файлом сделать из полного каталога выборку товаров стоимостью до 10000 и страной производства Россия 


# Реализация

### Настройка окружения и запуск

**1. Установить Python**

Версия 3.10 или выше.
В Убунте 24 он уже есть.
<br>Проверить версию:
```
python3 --version
```
**2. Установить виртуальное окружение для Питона**
```
sudo apt install python3-venv -y
```
**3. Склонировать репозиторий**
```
git clone git@github.com:OD1402/python.git
```
**5. Перейти в папку**
```
cd ~abc/src/python
```
**6. Создать виртуальное окружение**
```
python3 -m venv venv
```
После выполнения команды появится папка "venv/."

**7. Активировать виртуальное окружение**
```
source venv/bin/activate
```
В начале строки терминала должно появиться (venv).

**8. Перейти в папку с проектом**
```
cd wildberries_scan_scrapy
```
**9. Установить Python зависимости**
```
pip install --upgrade pip
pip install -r requirements.txt
```
**10. Запустить парсер**
```
scrapy crawl wildberries
```
